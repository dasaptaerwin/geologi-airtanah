# Slide Geologi Airtanah versi 2018
Repositori ini berisi slide kuliah Geologi Air Tanah versi 2018 yang akan diperbarui di tahun 2020. 
Format slide adalah ppt, tapi dibuat menggunakan LibreOffice. Jadi anda dapat pula membukanya dengan LibreOffice.
Nama slide diurutkan berdasarkan minggu `<urut_minggu>-<nama_kuliah>-<tahun>.ppt`.
